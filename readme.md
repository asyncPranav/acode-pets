# 🐾 Acode Pets Plugin

Bring your Acode editor to life with **interactive animated pets**!  
Cats and dogs that **walk, run, sleep, jump, and react while you code**, inspired by VS Code Pets.  

**Developed by [asyncpranav](https://github.com/asyncpranav)** ❤️

---

## ✨ Features

- 🐱 Cat & Dog pets  
- 🎨 Customizable skins & colors  
- 🎞️ Smooth animations: idle, walk, run, sleep, jump, sit, happy  
- 💤 Idle → Sleep, 🏃 Typing → Run  
- 😄 Mood system: Happy, Bored, Sleepy  
- 👆 Tap to interact (jump)  
- 📱 Mobile & desktop optimized  

---

## 🚀 Installation

1. Open **Acode** → **Extensions / Plugins**  
2. Search **Acode Pets** → Install & enable  
3. Restart Acode  


---

## 🛠️ Usage

- Pets appear on startup  
- Typing → pet runs  
- Idle → pet sleeps  
- Tap → pet jumps  

---

## ⚙️ Settings

Open **Ctrl + Shift + P** → Search **“Acode Pets”** → **Settings**  

Customize:  
- Pet type 🐾  
- Skin 🎨  
- Size & speed ⚡  
- Behavior 🧠  

---

## 🐾 Pet States

`idle` | `walk` | `run` | `sleep` | `jump` | `sit` | `happy`
